DOM Manipulation

DOM (Document Object Model) manipulation refers to the process of dynamically changing the content and appearance of a web page using JavaScript.

The DOM is a hierarchical representation of an HTML, where each element of the document is represented by a node in the tree-like structure. These nodes can be accessed and manipulated using JavaScript, allowing you to change the content, attributes, and style of the elements in real-time.

Common ways to manipulate the DOM using JavaScript:

  • Selecting elements: Use the querySelector or querySelectorAll methods to select elements on the page based on their ID, class, tag name, or other attributes.
  • Modifying element content: Use the innerHTML property to change the HTML content of an element, or the textContent property to change the text content.
  • Changing element attributes: Use the setAttribute and removeAttribute methods to add or remove attributes from an element, such as class, src, href, or style.
  • Adding and removing elements: Use the createElement, appendChild, and removeChild methods to create and add new elements to the DOM, or remove existing ones.
  • Modifying element style: Use the style property to change the CSS style of an element, such as its background color, font size, or border.
